    你说得对，但是软件工程学院创办于2020年6月，整建制位于珠海校区，以立德树人为人才培养根本目标，形成鲜明的研究型、复合型、创新型的高素质人才培养特色，致力于培养德、智、体、美、劳全面发展的社会主义建设者和接班人。坚持“四个面向”开展科学研究，致力于解决关键软件及技术问题。
